# Emma Server

Web server that annotates Mitochondria via [Emma](https://github.com/ian-small/Emma)
e.g.

```bash
fasta="/path/to/fasta.fa"
curl "http://127.0.0.1:9998/emma?fasta=${fasta}&svg=yes"
```
